..:: Instructions ::..

This is a Playground for The Chiming Sands event! This feature simulates the well-known event mechanic we know from the Amuni event!

You can interact with it – use provisions to purchase tools , use your tools to clear the ancient city and uncover rewards to help you progressing on the progress lane. Practising this minigame will help you to not only better understand the way this event works, but will also help you better navigate between the ancient walls and improve the efficiency of using your tools.

You can use the button to purchase more tools. Use the same button to get back to the game. This toolshop allows you to try various combinations of newly introduced packages (that are also the part of this year's event) and learn, which ones work the best for you!

You can use the Restart button to clear your current progress and start again. The game ends when all possible provisions are used and no more tools can be placed. In case you broke your personal record, it will get displayed on the page. This information is only visible to you and you can play again to try improve your strategies and beat your high score again.

Note that this is a Playground and it is only a practise tool. No rewards will be transferred to your game account when you collect them on my website. All assets used belong to InnoGames GmbH.
