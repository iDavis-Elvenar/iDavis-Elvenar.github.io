To complete the questline, you need a total of 24 Training Centers. Therefore, all of our layouts focus on reaching this goal. All layouts were created using the brand-new City Planner. You can visit it to experiment with your own layouts and find what suits you best!

  • This layout uses an ideal 5-5-1-1 approach, meaning each main building receives the maximum possible boost from the set. Training Centers that do not benefit from the respective main buildings are included only once, solely to support the main building itself.



  • Another 5-5-1-1 layout with a diagonal shape, in case this one fits your currently available space better.



  • A 3-3-3-3 layout. Note that while both main buildings still receive the maximum boost from the Training Centers, this layout provides less Intrinsic Motivation, as more Training Centers are connected to main buildings from which they do not receive any bonuses.



  • Another 3-3-3-3 layout with a horizontal shape, in case this one fits your currently available space better.



The layouts have been created using the Elvenar Tools website.
