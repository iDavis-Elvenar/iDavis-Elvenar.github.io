  • The production buildings of this settlement work similarly to the sets we know from events. In order to obtain the maximal production, each building needs to touch 3 different buildings of this set. When you build 4 production buildings of each type, you reach the maximal production boost available. You can use any of the following shapes based on your current city space.
    The portal is not included in these layouts for simplicity's sake, but know that you can place it anywhere that touches at least one of any of the ports.








The layouts have been created using the Elvenarchitect tool.
